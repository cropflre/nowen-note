# 微信收藏导入教程

Nowen Note 支持导入 [WeChatDataAnalysis](https://github.com/LifeArchiveProject/WeChatDataAnalysis) 导出的微信收藏 **JSON ZIP**，并迁移收藏正文、图片、视频、语音、文件、来源、时间和标签。

本教程重点说明两件事：

1. 如何从微信本地数据中获得 Nowen Note 可识别的微信收藏 ZIP；
2. 如何将该 ZIP 导入 Nowen Note。

> [!IMPORTANT]
> WeChatDataAnalysis 是第三方开源工具，不属于 Nowen Note。请只处理你本人有权访问的微信数据，并从其官方 GitHub Release 下载程序。

## 一、Nowen Note 需要什么文件

Nowen Note 需要的是：

```text
WeChatDataAnalysis
→ 收藏
→ 导出收藏
→ 格式选择 JSON
→ 工具生成的原始 .zip 文件
```

请注意：

- 需要选择 **收藏** 数据集，不是普通聊天记录；
- 导出格式必须选择 **JSON**；
- 最终文件通常是 `.zip`，不是单独的 `.json`；
- 不要导入 HTML、TXT 或 Excel 版本；
- 不要使用“账号数据归档”ZIP 代替收藏导出 ZIP；
- 不要解压后修改、重命名内部文件或重新压缩。

Nowen Note 同时兼容以下收藏 JSON 结构：

- `dataset: "favorites"` 且收藏位于 `items` 中的原始收藏数据；
- WeChatDataAnalysis 当前导出的聊天归档结构，其中收藏内容位于 `messages` 或 `conversations[].messages` 中。

## 二、准备工作

### 1. 准备 Windows 电脑

WeChatDataAnalysis 当前主要面向 Windows，官方 README 提供 Windows 安装包，并说明用于处理微信 4.x 的本地数据。

建议准备：

- 已在电脑上使用过微信 4.x 的 Windows 电脑；
- 足够的磁盘空间；
- 可以正常打开的本人微信收藏数据；
- 一个单独的导出目录，例如：

```text
D:\WeChat-Export\Favorites
```

收藏中如果包含大量图片、视频或文件，导出 ZIP 可能明显大于纯文本数据。

### 2. 下载 WeChatDataAnalysis

打开官方项目：

- 项目主页：<https://github.com/LifeArchiveProject/WeChatDataAnalysis>
- 最新版下载：<https://github.com/LifeArchiveProject/WeChatDataAnalysis/releases/latest>

在 Release 页面下载类似以下名称的安装包：

```text
WeChatDataAnalysis.Setup.<版本号>.exe
```

安装并启动程序。

> [!WARNING]
> 微信收藏包含大量个人隐私。不要从网盘、论坛或陌生群文件下载修改版程序，也不要把导出的 ZIP 随意发送给他人。

## 三、让 WeChatDataAnalysis 读取微信数据

首次启动后，按照 WeChatDataAnalysis 界面提示加载你自己的微信账号数据。

由于上游工具的账号识别和数据加载界面可能随版本调整，本教程不固定描述密钥或目录按钮的位置。完成的判断标准是：

1. 工具中已经选中了正确的微信账号；
2. 左侧可以进入“收藏”页面；
3. 收藏页面顶部能显示收藏总数；
4. 页面中能看到实际收藏内容。

如果电脑上存在多个微信账号，请先确认当前选中的账号正确，再执行导出。

如果收藏页面为空，可以先：

- 检查是否选错微信账号；
- 点击收藏页面右上角的刷新按钮；
- 确认当前电脑中确实存在对应账号的微信 4.x 本地数据；
- 确认收藏内容在微信客户端中仍然可以访问。

## 四、导出微信收藏 JSON ZIP

### 第 1 步：打开“收藏”页面

在 WeChatDataAnalysis 左侧导航栏中点击：

```text
收藏
```

页面顶部会显示类似：

```text
共 1234 条收藏
```

### 第 2 步：决定导出范围

直接导出全部收藏时，不需要设置筛选条件。

需要分批导出时，可以使用：

- 收藏页面顶部的关键词搜索；
- 收藏类型筛选；
- 导出弹窗中的内容类型选择。

关键词搜索会影响本次导出的记录范围。收藏很多或 ZIP 过大时，可以按类型或关键词分批导出。

### 第 3 步：点击“导出收藏”

点击收藏页面右上角的导出图标，按钮提示为：

```text
导出收藏
```

随后会打开“导出收藏”设置弹窗。

### 第 4 步：把格式改成 JSON

在“格式与内容”区域中选择：

```text
JSON
```

> [!CAUTION]
> WeChatDataAnalysis 当前打开导出弹窗时默认选择的是 **HTML**。必须手动点击 **JSON**，否则生成的包不能作为 Nowen Note 微信收藏导入包使用。

不要选择：

- HTML
- TXT
- Excel

### 第 5 步：选择收藏内容类型

推荐首次迁移时保持全部类型选中，包括：

- 文本；
- 图片；
- 语音；
- 视频；
- 文件；
- 链接 / 视频号；
- 位置；
- 聊天记录；
- 表情；
- 其他。

只想迁移部分内容时，可以取消不需要的类型。

至少需要选择一种内容类型，否则无法开始导出。

### 第 6 步：设置文件名和保存目录

在“文件与目录”区域：

1. “文件名”可以留空，工具会自动生成；
2. “保存目录”是必选项；
3. 点击“选择目录”，选择一个空间充足的位置。

建议使用容易识别的文件名，例如：

```text
微信收藏-全部-2026-07-16
```

不需要手动添加 `.zip` 后缀。

### 第 7 步：开始导出

点击：

```text
开始导出
```

完成后，界面会显示：

```text
已导出 N 条：<输出文件路径>
```

请打开该输出路径，找到生成的 `.zip` 文件。

### 第 8 步：不要修改生成的 ZIP

当前 WeChatDataAnalysis 会把结构化 JSON 和能够找到的本地媒体一起写入 ZIP。通常不需要单独开启“导出媒体”选项。

请直接保留工具生成的原始 ZIP：

- 不要解压后只拿出 JSON；
- 不要删除媒体目录；
- 不要修改内部 JSON；
- 不要重新压缩；
- 不要只把 ZIP 改名为其他扩展名。

修改包内路径或文件名可能导致图片、视频、语音和文件无法匹配。

## 五、检查导出的文件是否正确

导入 Nowen Note 前，可以先检查以下项目。

### 正确文件应满足

- 文件扩展名为 `.zip`；
- 来源是 WeChatDataAnalysis 的“收藏”页面；
- 导出格式选择了 JSON；
- 导出完成时显示的记录数不为 0；
- ZIP 文件大小不是 0 KB。

### 可选的只读检查

可以使用压缩软件查看 ZIP 内部文件列表，但不要修改或重新保存。

当前版本的导出包通常可以看到：

- 一个或多个 `messages.json`；
- 收藏相关的结构化数据；
- 图片、视频、语音或文件资源目录。

内部目录名称可能随 WeChatDataAnalysis 版本变化，只要它是工具直接生成的收藏 JSON ZIP，一般无需手动调整。

### 以下文件不能直接使用

```text
收藏.html
收藏.txt
收藏.xlsx
普通聊天记录.zip
账号数据归档.zip
手动创建的“只有一个 JSON”的 ZIP
```

## 六、导入 Nowen Note

### 第 1 步：打开微信收藏入口

进入：

```text
设置
→ 数据管理
→ 导入
→ 第三方数据迁移
→ 微信收藏
```

微信收藏现在是独立的一级导入来源，不需要先进入“有道云笔记”。

### 第 2 步：选择 ZIP

点击：

```text
选择微信收藏 ZIP
```

选择 WeChatDataAnalysis 刚刚生成的原始 ZIP。

ZIP 会上传到当前 Nowen Note 实例的后端进行安全预检和导入。大型媒体包不会在浏览器内完整解压。

### 第 3 步：查看预检结果

预检完成后，重点检查：

- 识别到的收藏总数；
- 各收藏类型数量；
- 日期范围；
- 标签数量；
- 媒体引用数；
- 已找到和缺失的媒体数量；
- 重复收藏数量；
- 安全或兼容性警告。

如果预检数量与 WeChatDataAnalysis 的导出数量差异很大，不要急着正式导入，先检查导出格式和筛选条件。

### 第 4 步：配置导入选项

#### 根笔记本名称

默认为：

```text
微信收藏
```

可以改为：

```text
微信收藏备份
```

或：

```text
微信收藏 2026
```

#### 按年份创建子笔记本

开启后，默认生成类似结构：

```text
微信收藏
├── 2026
├── 2025
└── 其他年份
```

收藏很多时建议开启。

#### 保留微信标签

开启后，微信收藏标签会创建或复用为 Nowen Note 标签。

#### 媒体缺失时继续导入正文

建议保持开启。

当某张图片、视频、语音或文件在本地微信数据中已经不存在时，Nowen Note 仍会导入正文、标题、来源和时间，并在报告中记录媒体缺失。

#### 重复数据处理

- **跳过**：默认选项，已导入的收藏不会重复创建；
- **更新**：更新原笔记，适合补齐后来找回的媒体；
- **创建副本**：始终新建一篇笔记。

首次导入建议使用“跳过”。

#### 导入类型

可以进一步选择只导入某些收藏类型。

如果导出时已经做过类型筛选，这里通常保持全部选中即可。

### 第 5 步：开始导入

点击：

```text
开始导入
```

导入期间不要重复点击按钮，也不要关闭或刷新页面。

### 第 6 步：检查导入报告

导入完成后检查：

- 新增；
- 更新；
- 跳过；
- 部分成功；
- 失败；
- 附件写入；
- 媒体缺失；
- 标签创建和复用。

单条收藏或单个附件失败不会中断整批导入。

## 七、内容如何映射到 Nowen Note

- 文本和微信笔记转换为可编辑富文本；
- 图片和表情进入附件系统，并在正文中显示；
- 视频、语音和普通文件进入附件系统；
- 网页、音乐、商品、视频号和小程序保留安全链接；
- 位置信息保留地点、地址和经纬度；
- 合并聊天记录转换为结构化正文；
- 来源名称、收藏类型和收藏时间写入笔记；
- 微信收藏标签转换为 Nowen Note 标签；
- 默认按“微信收藏 / 年份”创建目录。

## 八、增量更新收藏

后续新增微信收藏后，可以重新从 WeChatDataAnalysis 导出一份新的完整 JSON ZIP，再导入 Nowen Note。

Nowen Note 会使用微信收藏的 `serverId`、`localId` 或稳定内容摘要判断来源。

推荐策略：

```text
日常增量备份：跳过
内容或媒体有变化：更新
需要保留两个版本：创建副本
```

来源映射按用户和工作区隔离。同一收藏可以分别导入个人空间和不同工作区。

## 九、常见问题

### 1. WeChatDataAnalysis 收藏页面没有内容

依次检查：

1. 是否选择了正确的微信账号；
2. 是否点击了收藏页面的刷新按钮；
3. 当前电脑是否存在该账号的微信 4.x 本地数据；
4. 微信客户端中是否仍能打开这些收藏；
5. WeChatDataAnalysis 是否为官方 GitHub Release 的较新版本。

### 2. 导出按钮无法点击

导出弹窗需要同时满足：

- 已选择微信账号；
- 已选择保存目录；
- 至少选择一种内容类型；
- 当前没有正在执行的导出任务。

### 3. Nowen Note 提示“未识别到微信收藏数据”

最常见原因是导出格式选错。

重新导出，并确认：

```text
数据集：收藏
格式：JSON
结果：工具生成的原始 ZIP
```

WeChatDataAnalysis 的导出弹窗默认是 HTML，请特别检查是否已经手动切换为 JSON。

### 4. 导出了一个 JSON，为什么不能导入

当前推荐流程生成的是包含 JSON 和媒体的 ZIP。

不要只拿出 JSON，也不要自行创建 ZIP。请直接选择 WeChatDataAnalysis 导出完成后显示的 `.zip` 文件。

### 5. 媒体缺失但正文成功导入

说明导出 JSON 中存在媒体引用，但导出工具没有在本地数据中找到对应资源。

可以尝试：

1. 在微信客户端中确认原图片、视频或文件仍可打开；
2. 回到 WeChatDataAnalysis 刷新收藏；
3. 重新导出 JSON ZIP；
4. 在 Nowen Note 中使用“更新”策略再次导入。

如果原媒体已经被微信清理或不再可访问，通常只能保留正文和媒体缺失提示。

### 6. ZIP 太大或上传失败

可以在 WeChatDataAnalysis 中分批导出：

- 按收藏类型导出；
- 使用关键词搜索后导出；
- 将视频和普通文件单独导出；
- 确认 Nowen Note 服务端的上传和附件大小限制。

多个分批 ZIP 可以依次导入，默认“跳过”策略会减少重复笔记。

### 7. 再次导入会不会生成重复笔记

默认使用“跳过”，一般不会重复创建。

需要补齐媒体或更新内容时选择“更新”。

### 8. 可以直接从手机微信导入吗

目前不支持。

Nowen Note 不直接读取手机微信数据，也不会在浏览器中提取微信数据库密钥。当前流程需要先用 WeChatDataAnalysis 从本人电脑上的微信本地数据导出收藏 JSON ZIP。

### 9. macOS 可以导出吗

WeChatDataAnalysis 当前官方快速开始主要提供 Windows EXE，并面向微信 4.x 本地数据。建议在保存有本人微信数据的 Windows 电脑上完成导出，再将 ZIP 上传到 Nowen Note。

### 10. 导入到哪个空间

微信收藏会导入到当前在“数据管理”中选择的个人空间或工作区。

- 工作区导入需要相应编辑权限；
- 个人空间导入受管理员的个人导入开关控制；
- 导入前请确认顶部空间范围选择正确。

## 十、隐私与安全建议

微信收藏 ZIP 可能包含：

- 私人聊天内容；
- 图片和视频；
- 语音；
- 文件；
- 联系人名称；
- 地点和时间信息。

建议：

- 只从官方 GitHub Release 下载 WeChatDataAnalysis；
- 只处理本人有权访问的数据；
- 不把导出的 ZIP 上传到不受信任的网站；
- 导入完成后删除临时副本，或放入加密磁盘；
- 使用远程部署的 Nowen Note 时，确认服务器和传输链路可信；
- 不在 Issue、聊天群或日志中公开真实收藏 ZIP。

Nowen Note 会对 ZIP 执行路径穿越、压缩比、文件数量、总体积、危险文件和 JSON 深度等安全检查，但这不能替代对导出文件本身的妥善保管。
