# Nowen Note Web Clipper — 隐私权政策

**最后更新日期：2025 年 4 月 24 日**

Nowen Note Web Clipper（以下简称"本扩展"）是一款浏览器扩展，用于将网页内容剪藏到用户自托管的 Nowen Note 笔记服务。我们重视您的隐私，本政策说明本扩展如何处理您的数据。

## 1. 数据收集与使用

本扩展**仅在用户主动触发剪藏操作时**收集以下数据：

- **网页内容**：当前页面的正文文字、图片、HTML 结构，用于生成笔记内容。
- **网页元信息**：当前页面的标题和 URL，用于标注笔记来源。
- **屏幕截图**：仅在用户选择截图模式时，截取当前页面的可视区域或整页长图。

以上数据**仅用于**完成剪藏功能，即将内容保存到用户自己配置的 Nowen Note 服务器。

## 2. 用户配置数据

本扩展在浏览器本地（chrome.storage）存储以下配置信息：

- Nowen Note 服务器地址
- API Token（用于身份验证）
- 默认笔记本名称、标签、输出格式等偏好设置

这些数据**仅存储在用户浏览器本地**，不会上传到任何第三方服务器。

## 3. 数据传输

本扩展收集的网页内容和截图**仅发送到用户自行配置的 Nowen Note 服务器**。本扩展：

- ❌ **不会**将数据发送到开发者或任何第三方服务器
- ❌ **不会**收集、存储或传输用户的个人身份信息
- ❌ **不会**追踪用户的浏览历史或浏览行为
- ❌ **不会**在后台自动收集任何数据
- ❌ **不会**出售或共享用户数据

## 4. 第三方服务

本扩展不使用任何第三方分析、广告或追踪服务。所有数据处理均在用户浏览器本地完成，数据传输目标完全由用户自行控制。

## 5. 权限说明

| 权限 | 用途 |
|------|------|
| activeTab | 访问用户主动触发剪藏时的当前标签页 |
| scripting | 注入内容提取脚本以获取网页正文 |
| contextMenus | 在右键菜单中提供剪藏快捷入口 |
| storage | 在浏览器本地保存用户配置 |
| notifications | 在截图等场景下通知用户操作结果 |
| debugger | 仅用于整页长截图功能（CDP 协议） |
| host_permissions | 支持在任意网页上执行剪藏和下载图片 |

所有权限仅在用户主动操作时使用，不会在后台自动运行。

## 6. 数据安全

- 用户的 API Token 仅存储在浏览器本地 chrome.storage 中
- 与 Nowen Note 服务器的通信建议使用 HTTPS 加密连接
- 本扩展不包含任何远程加载的代码

## 7. 用户控制

- 用户可随时在扩展选项页修改或删除已保存的配置
- 卸载扩展将自动清除所有本地存储的数据
- 用户完全控制数据的传输目标（自托管服务器地址）

## 8. 政策变更

如本隐私政策发生变更，我们将在此页面更新内容和日期。
